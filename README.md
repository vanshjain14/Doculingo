## Tech Stack
- **Frontend:** React, Tailwind
- **Backend:** Flask

- **Vector DB:** FAISS / Pinecone / Chroma
- **LLMs:** OpenAI + custom model

## Installation

### Backend
```cd backend
pip install -r requirements.txt
python app.py
```
### Frontend
```
cd frontend
npm install
npm start
```
